Common questions about the Track 02 challenge — reach out if anything else comes up.

Cloud platforms underpin modern construction workflows, but because standards vary they can consume large amounts of energy and be exposed to cybersecurity risk. This track aims to tackle both at once.
Cloud environments built by different teams to different standards are hard to govern consistently — they leave security gaps and run plenty of high-carbon, inefficient workloads, wasting energy and driving up cost.
Build a solution that continuously monitors security vulnerabilities in cloud environments while identifying high-carbon or low-efficiency workloads, enabling safer and more energy-efficient cloud operations.
Yes. An ideal solution lowers energy use and carbon footprint while maintaining security, provides automated alerts and smart optimization recommendations, and continuously optimizes the system for better resource efficiency.
Higher cloud security, lower carbon footprint and cost, better resource efficiency, and stronger support for construction workflows with higher overall operational efficiency.
