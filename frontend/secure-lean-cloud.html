Secure & Lean Cloud

Classify cloud security risks, calculate security posture, review findings, and visualize risky resource relationships.

This feature analyzes cloud resources, identifies misconfigurations, identity risks, vulnerabilities, and resource waste, then presents the results through dashboards, findings, and a security graph.
