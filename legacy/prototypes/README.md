# Legacy Prototypes

This folder keeps earlier prototype scripts and experimental files that are not required for the current SecEff demo run.

The active application entry point is:

```text
backend/server.js
```

The active frontend is served from:

```text
http://localhost:5000/frontend/index.html
```

Files in this folder are retained for reference only. They should not be used as the main startup path unless a future refactor intentionally restores them.
